Hytousuce、WindSpiritSR

就在几小时前，我成功部署了自己的博客。作为博客的第一篇文章，在这里记录一下我从萌生创建博客的想法，到付诸行动，直至完成部署的全过程。

第一次写相关的内容，行文有点拖沓，或者读起来有很多难受的地方，还请见谅 uwu

萌生想法

大一下时，我接触了一点点皮毛就什么都想试试，即使是在完全没弄清楚“这东西是啥”的情况下，主打又菜又想玩。

2023 年 4 月，网上冲浪的我，浏览了一些大佬的个人博客后，有了创建属于自己的个人博客的想法（被花里胡哨的 CSS 迷惑了双眼）。首次对外提出这个想法是在与 Hytousuce 的聊天中，随后 Hytou 推荐了 HEXO 博客框架。当时我浅浅看了一下文档，安装好了Git和Node.js做了一些准备。

然后半期考试来了。

在各学科总分里，半期考试和任务的占比还算高，可耽误不得。于是搭建博客的事情被搁置了。

“之后再来吧。”

直到 5 天前……

再次行动

2023 年 11 月 5 日，也是和 Hytousuce 聊天的过程中，谈到了关于域名的事情。这段时间兴起购买原神相关域名并解析到自己大学的行为（比如中国原神大学 ）

随后 Hytou 向我展示了在阿里云的域名列表，引起了我的注意。我想是否能在里面找到有自己的特征的域名。

我的昵称 Kurotani Takeo，简化->KuroTake，再简化->krtk

搜索一下：

域名 价格/年
krtk.top ￥ 26
krtk.co ￥ 1349
krtk.cn ￥ 1691
krtk.xyz ￥ 8156
krtk.com ￥ 86390
…… ……

以一只带学生的财力，相信目标一目了然。短域名和价格让我一眼相中了“krtk.top”

Hytou：“买了吧，过了这村没这店”
Hytou：“如果呢接受 我觉得买了比较好”
Hytou：“四个字母的域名挺有价值了”
Hytou：“我觉得买了好，毕竟 26 不算贵”

然后就买了。

买了域名我能做什么？

我想起来半年前的想法：要不我搭建个人博客吧？

动手

大二上，一学期最忙的三周半总算过去了。没有作业和任务，绝佳的机会。

从昨晚 11 点直到刚才，我一直在调整博客的配置。其过程太过于头疼和令人高血压（主要是因为本人在这方面过于无知），于是略去大部分步骤简单讲讲：

HEXO 安装，跟着官网文档 和知乎的教程 做完了准备工作。

第一个问题出在域名的解析上。也许是校园网的缘故，我获取到的 GitHub Pages 的 IP 地址一直是 198.18.0.xx 的内网 IP……（我还拿去解析了草）（最后由 Hytou 帮忙 Ping 出了正确的 IP 地址）

随后，krtk.top 能访问了，但是 CSS 并未起作用……后来才发现我照着 config 的注释，在 url 填写了 KurotaniTakeo.github.io/project，而事实是根本不用加上最后那个/project……

最后，找到了个米米的 HEXO theme，但是也许是碰到了 theme 配置文件中一些不该碰的东西，于是 CSS 再次失效了……

直到刚才，大部分内容才准备妥当。

结尾

这就是这个博客从想法到实际的整个过程（结尾得好突然）

本想发表一些自认为很深刻的感想，但是想着这篇又臭又长的第一篇博客（日记？）已经烂了个彻底了，于是删掉了那些自己看了都像是为了完成目的而写的总结。何况看到这儿的人也不多，随便写写便是（

还是有一些成就感的。毕竟在此之前，我一直认为搭建博客是在我大学毕业之后才会去尝试的事。

希望我能坚持把博客写下去吧——
